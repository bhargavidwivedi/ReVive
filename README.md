<br/>

Hospital readmissions within 30 days of discharge are one of the most critical and costly challenges in healthcare:

&nbsp;&nbsp;&nbsp; `COST` &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Over **$26 billion** spent annually on preventable readmissions in the US alone

&nbsp;&nbsp;&nbsp; `SIGNAL` &nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Readmissions indicate gaps in post-discharge care or premature discharge

&nbsp;&nbsp;&nbsp; `IMPACT` &nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Disproportionately affects patients with diabetes, heart failure & COPD

&nbsp;&nbsp;&nbsp; `SOLUTION` &nbsp;&nbsp; **ReVive** provides clinicians a real-time risk score per patient — enabling proactive care planning before discharge

```
│   └── predictor/
│       ├── views.py            ← /predict endpoint logic
│       └── serializers.py
│
├── frontend/                   ← Streamlit UI
│   └── app.py                  ← Clinician dashboard
│
├── requirements.txt
└── README.md
```

**Data Flow:**

```
Clinician Input (Streamlit)
        ↓
  HTTP POST → /predict
        ↓
  Preprocessing Pipeline
        ↓
  XGBoost Model Inference
        ↓
  Risk Score + Flag Returned
        ↓
  Result on Dashboard
```

<br/>

**`STEP 01`** &nbsp; Clone the repository

```bash
git clone https://github.com/bhargavidwivedi/ReVive.git
cd ReVive
```

**`STEP 02`** &nbsp; Create & activate virtual environment

```bash
python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate
```

**`STEP 03`** &nbsp; Install dependencies

```bash
pip install -r requirements.txt
```

**`STEP 04`** &nbsp; Run the Django backend

```bash
cd backend
python manage.py migrate
python manage.py runserver
# API live at → http://127.0.0.1:8000/
```

**`STEP 05`** &nbsp; Run the Streamlit frontend *(new terminal)*

```bash
cd frontend
streamlit run app.py
# Dashboard live at → http://localhost:8501/
```

<!-- API -->
<img width="100%" src="https://capsule-render.vercel.app/api?type=rect&color=000000&height=18&text=%E2%9C%A6%20%20API%20REFERENCE&fontSize=13&fontColor=C9A84C&fontAlign=5&fontAlignY=65"/>

<br/>

&nbsp;&nbsp;&nbsp; `ENDPOINT` &nbsp;&nbsp; `POST /predict/`

**Request:**

```json
{
  "age": 67,
  "gender": "Female",
  "diagnosis": "Heart Failure",
  "num_prior_admissions": 3,
  "length_of_stay": 7,
  "num_medications": 12,
  "discharge_type": "Home",
  "comorbidity_score": 4
}
```

**Response:**

```json
{
  "readmission_probability": 0.78,
  "risk_level": "High",
  "flag": true,
  "message": "Patient is at HIGH risk. Early intervention recommended."
}
```

<div align="center">

| Field | Type | Description |
|:------|:----:|:------------|
| `readmission_probability` | `float` | Score between 0 and 1 |
| `risk_level` | `string` | `Low` · `Medium` · `High` |
| `flag` | `boolean` | `true` if probability > 0.5 |
| `message` | `string` | Clinical recommendation |

</div>
